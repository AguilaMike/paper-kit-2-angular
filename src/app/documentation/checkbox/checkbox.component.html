Checkboxes


Usage can be done via Data Attributes: 

<div class="checkbox">
   <input id="checkbox1" type="checkbox">
   <label for="checkbox1">
       Unchecked
   </label>
</div>
<div class="checkbox">
  <input id="checkbox2" type="checkbox" checked="">
  <label for="checkbox2">
       Checked
   </label>
</div>
<div class="checkbox">
   <input id="checkbox3" type="checkbox" disabled="">
   <label for="checkbox3">
       Disabled unchecked
   </label>
</div>
<div class="checkbox">
  <input id="checkbox4" type="checkbox" checked="" disabled="">
  <label for="checkbox4">
      Disabled checked
  </label>
</div>
