Radio Buttons

To use the custom radio buttons, you need to import a separate Javascript file called 'paper-kit-radio.js'. 

<div class="form-check-radio">
  <label class="form-check-label">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1" >
    Radio is off
    <span class="form-check-sign"></span>
  </label>
</div>
<div class="form-check-radio">
  <label class="form-check-label">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios2" value="option2" checked="">
    Radio is on
    <span class="form-check-sign"></span>
  </label>
</div>
<div class="form-check-radio disabled">
  <label class="form-check-label">
    <input class="form-check-input" type="radio" name="exampleRadios1" id="exampleRadios3" value="option3" disabled="">
    Disabled Radio is off
    <span class="form-check-sign"></span>
  </label>
</div>
<div class="form-check-radio disabled">
  <label class="form-check-label">

  </label>
</div>
